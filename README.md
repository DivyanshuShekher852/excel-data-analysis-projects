# 🚦 Bihar Road Accident Dashboard (Excel)

## 📌 Project Overview
This project is an interactive Excel dashboard built to analyze road accident data in Bihar. The dashboard helps users understand accident trends, identify high-risk districts, and explore factors contributing to road accidents using Pivot Tables, Pivot Charts, and Slicers.

This project was created to improve my Excel and data analysis skills as part of my Data Analyst portfolio.

---

## 🎯 Objectives
- Analyze road accident patterns.
- Identify accident hotspots.
- Compare accident severity across districts.
- Monitor monthly and yearly accident trends.
- Build an interactive dashboard for better decision-making.

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Excel Formulas

---

## 📊 Dashboard Features
- Total Accidents
- Total Fatalities
- Total Injuries
- Monthly Accident Trend
- District-wise Accident Analysis
- Road Type Analysis
- Weather Condition Analysis
- Vehicle Type Distribution
- Severity Analysis
- Interactive Slicers for filtering data

---

## 📂 Dataset
The dataset contains road accident records with information such as:
- Date
- District
- Road Type
- Vehicle Type
- Weather
- Severity
- Number of Injuries
- Number of Deaths

---

## 📈 Key Insights
- Identified districts with the highest number of accidents.
- Compared accident frequency across different road types.
- Analyzed accident trends over time.
- Examined how weather conditions affect accident rates.
- Highlighted the distribution of accident severity.
